# 🌱 Smart Irrigation System

An IoT-based smart irrigation system using ESP32, Firebase, Flutter, and AI.

## 🚀 Features
- Real-time sensor monitoring
- Remote pump control
- Firebase cloud integration
- AI-based irrigation recommendation

## 🧠 Technologies Used
- ESP32
- Firebase Realtime Database
- Flutter
- Groq AI API

## ⚙️ Hardware Components
- DHT11 Sensor
- Soil Moisture Sensor
- LDR Sensor
- Relay Module
- Water Pump

## 📱 Mobile App
Displays sensor data and controls the irrigation system.

## 🔌 System Architecture
ESP32 → Firebase → Flutter App → User Control
